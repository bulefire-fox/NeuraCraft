### CyberFurry
但这也不是什么坏事,因为使用 `CyberFurry` 时十分省心,并且

- 提前训练的 `Furry` 专用模型,表象如同真正的 furry
- 完美的 `语c` 能力
- 无限制的使用 API (只要申请下来
- 快速响应,并且不会出现 `AI` 模型超时

但缺点也是不可避免的

- `CyberFurry` 系列API申请繁琐
- only for furry 导致其通用能力欠缺
- 可能对英文的支持不够好

尽管如此,我们目前还是推荐使用 `CyberFurry` 模型以获得还算优秀和省心的体验.最重要的是
```text
不用爆金币
```
