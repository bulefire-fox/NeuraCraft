# Getting Started with NeuraCraft
NeuraCraft is a Minecraft mod that adds AI players to the game.  
You can now chat with AI players in-game. We plan to make AI become actual player entities that can play alongside you in v4.0.

Follow these steps to begin your AI journey:

1. Install the `NeuraCraft` mod
2. Configure the `NeuraCraft` mod
   1. Choose your preferred `AI` model
   2. Obtain the corresponding `API endpoint` and `token`
   3. Enter the information in the configuration file
3. Launch the game and create a chat room using the chat room system
4. Use `AI <message>` to communicate with AI players
5. Enjoy the AI-powered experience

Detailed instructions for each step are provided below.

## Installing the `NeuraCraft` Mod
Download the `NeuraCraft` mod from your preferred platform and install it in your game's `mods` directory.<br>
Supported platforms:
- [Curseforge](https://www.curseforge.com/minecraft/mc-mods/neuracraft) <br>
- [Modrinth](https://modrinth.com/mod/neuracraft)<br>
- [GitHub Release](https://github.com/bulefire-dream/NeuraCraft/releases)

## Configuring the `NeuraCraft` Mod
The NeuraCraft mod offers various configuration options in its config file.<br>
### Choosing an AI Model
[Select an AI model that suits your needs](../API/choose/en_us.md)

### Obtaining API Credentials
Refer to the documentation links provided in [Selecting an AI Model](../API/choose/en_us.md)

### Entering Configuration Information
Fill in the required information in the config file. See [Selecting an AI Model](../API/choose/en_us.md) for detailed instructions.

## Creating a Chat Room In-Game
Access the chat room system in-game and create a chat room. Refer to [Chat Room System](../chatRoomManagerUse/en_us.md) for details.

## Using `AI <message>` to Chat
Example:
```text
AI Hello
```

## Enjoy the AI Experience
Try chatting with AI about something different? You might get unexpected replies!
