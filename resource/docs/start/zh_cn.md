# 开始使用 NeuraCraft
NeuraCraft 是一个 Minecraft 模组，它添加了 Agent 到游戏中。
你现在可以在游戏中与 Agent 对话，我们计划在 v4.0 到时候让 Agent 真正变成一个玩家实体并和你一起游戏，

你可以遵循以下步骤开启你的 Agent 之旅

1. 安装 `NeuraCraft` 模组
2. 配置 `NeuraCraft` 模组
   1. 调整配置文件至你所希望的模式
   2. 装载插件以获取额外的模型支持 （可选）
3. 进入游戏并配置使用 Agent 系统创建 Agent
4. 输入 `AI <message>` 来与 `AI玩家` 对话
5. 尽情享受 AI 带来的乐趣

接下来会提供每个步骤的详细指导

## 安装 `NeuraCraft` 模组
在你喜欢的平台上上下载 `NeuraCraft` 模组，并安装到你的游戏目录的`mods`中。<br>
我们支持的平台有:
- [Curseforge](https://www.curseforge.com/minecraft/mc-mods/neuracraft) <br>
- [Modrinth](https://modrinth.com/mod/neuracraft)<br>
- [GitHub Release](https://github.com/bulefire-dream/NeuraCraft/releases)

## 配置 `NeuraCraft` 模组
NeuraCraft 模组提供了多种配置选项，你可以在配置文件中进行配置。<br>


## 进入游戏并配置使用聊Agent系统创建 Agent
在游戏内进入Agent系统，并创建一个Agent，具体请参阅[Agent系统](../chatRoomManagerUse/zh_cn.md)

## 输入 `AI <message>` 来与 `AI玩家` 对话
示例:
```text
AI 你好
```
## 尽情享受 AI 带来的乐趣
尝试与 Agent 聊点不一样的? 也许你会得到预期之外的回复呢!
