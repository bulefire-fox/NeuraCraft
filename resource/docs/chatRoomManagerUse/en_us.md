# Using NeuraCraft Chat Room System
The NeuraCraft chat room system allows players to manage multiple chat rooms and communicate within the game using in-game commands.

## Creating a Chat Room
Use 
```
/chatroom create <name>
```
command. <br>
The created chat room will automatically add the player.

## Deleting a Chat Room
Use 
```
/chatroom delete <name>
```
command. <br>
Players will automatically leave deleted chat rooms.

## Joining a Chat Room
Use 
```
/chatroom join <name>
```
command.

## Leaving a Chat Room
Use 
```
/chatroom exit <name>
```
command. <br>
NOTE: In current version `<name>` must match the chat room the player is currently in.
