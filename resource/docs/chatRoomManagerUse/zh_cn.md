# 使用 NeuraCraft 的聊天室系统
NeuraCraft 的聊天室系统允许玩家在游戏中进行多聊天室的管理和聊天, 并使用命令进行游戏内对聊天室的操作。
## 创建聊天室
创建聊天室可以使用 `/chatroom create <聊天室名称>` 命令。 <br>
创建聊天室后，聊天室将自动加入玩家。

## 删除聊天室
删除聊天室可以使用 
```
/chatroom delete <聊天室名称>
```
命令。
玩家会自动退出已删除的聊天室

## 加入聊天室
加入聊天室可以使用 
```
/chatroom join <聊天室名称>
```
命令。

## 退出聊天室
退出聊天室可以使用 
```
/chatroom exit <聊天室名称>
``` 
命令。
注意当前版本 `<聊天室名称>` 必须与玩家所在的聊天室相同

## 寻找自己在哪个聊天室
寻找自己在哪个聊天室可以使用 
```
/chatroom find
```
命令。
